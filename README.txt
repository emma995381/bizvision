========================================
  BizVision – Modern Responsive Business HTML Template
========================================

Thank you for purchasing BizVision!
This template is built with HTML, CSS, and JavaScript for a professional business website with a responsive layout and auto-sliding image slider.

----------------------------------------
📂 PACKAGE CONTENTS
----------------------------------------
- index.html (Homepage)
- services.html (Services page)
- about.html (About page)
- contact.html (Contact page)
- faq.html (FAQ page)
- /css/ (Stylesheets)
- /js/ (JavaScript files)
- /images/ (Image assets)
- /Preview/ (Preview images & promo banner)
- README.txt (This file)

----------------------------------------
💻 FEATURES
----------------------------------------
✔ Fully responsive layout (desktop, tablet, mobile)
✔ Auto-sliding hero image slider
✔ Fixed top navigation bar
✔ SEO-friendly, lightweight code
✔ Easy to customize
✔ Cross-browser compatible

----------------------------------------
⚙️ INSTALLATION
----------------------------------------
1. Upload all files & folders to your web hosting server.
2. Open index.html in a browser to view the site.

----------------------------------------
🎨 CUSTOMIZATION
----------------------------------------
- Change slider images in /images/slider/ (or update code in index.html if filenames differ).
- Update colors & fonts in /css/styles.css.
- Edit text directly inside the HTML files with a code editor.

----------------------------------------
🌐 LIVE DEMO (GitHub Pages)
----------------------------------------
1) Create a repo named 'bizvision'.
2) Upload all files from this package.
3) Settings → Pages → Source: main branch, root (/).
4) Your demo will be available at: https://YOUR-USERNAME.github.io/bizvision
